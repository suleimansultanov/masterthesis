# Designated Literature Sources (Citation Rule)

> **Rule:** Only cite from designated, peer-reviewed academic sources. Do **not** cite arXiv preprints (or other non-peer-reviewed preprint servers) unless a published, peer-reviewed version of the same work exists — in which case cite that version.

## Allowed sources

- **IEEE** (IEEE Xplore)
- **ACM** (ACM Digital Library)
- **ScienceDirect / Elsevier**
- **Scopus** (indexed)
- **Web of Science** (indexed)
- **Springer** (incl. SpringerLink, Nature Publishing Group)
- **SAGE**
- **MDPI**
- ...and other recognized, peer-reviewed publishers / indexed venues (e.g. JMLR, ICML/NeurIPS proceedings, Frontiers, PLOS, university presses such as MIT Press).

## Not allowed

- arXiv and other preprint servers (unless a peer-reviewed published version is cited instead)
- Blog posts, vendor whitepapers, non-peer-reviewed web pages

## Notes

- When adding a reference, record the venue and verify it is indexed/peer-reviewed before citing.
- Mark any entry whose authors/DOI/venue are not fully verified as **(unverified)** until checked.
