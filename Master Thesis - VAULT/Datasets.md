# Datasets

Два публичных бенчмарка, покрывающих комплементарные IoT healthcare сценарии (баланс breadth vs feasibility).

## OPPORTUNITY
- **Тип**: body-worn + ambient сенсоры.
- **Описание**: мультимодальные данные для activity recognition; **113 признаков** с accelerometers, gyroscopes, ambient sensors; включает естественный sensor dropout.
- **Релевантность**: стресс-тест высокоразмерной мультимодальной fusion при реалистичном шуме и dropout'е.
- **Источник**: Chavarriaga et al., 2013 — *Pattern Recognition Letters* (Elsevier). → [[References]]

## CASAS (Smart Home)
- **Тип**: ambient сенсоры.
- **Описание**: motion / door / temperature сенсоры для мониторинга активности пожилых в реальных smart-home средах.
- **Релевантность**: ambient elderly monitoring с гетерогенными типами сенсоров и temporal-паттернами активности.
- **Источник**: Cook et al., 2013 — *IEEE Computer*. → [[References]]

> [!note]
> Оба публичны, широко используются в IoT/ubicomp, позволяют прямое сравнение с прошлыми работами (IEEE, ACM, Springer).
